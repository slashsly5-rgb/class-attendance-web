<task type="checkpoint:human-action">
  <name>Create Supabase project via web dashboard</name>
  <action>Create Supabase project (requires web UI interaction)</action>
  <instructions>
1. Visit https://supabase.com and sign in (or create account if needed)
2. Click "New Project" button
3. Fill in project details:
   - **Name**: class-attendance (or your preferred name)
   - **Database Password**: Generate strong password (save in password manager!)
   - **Region**: Select closest region to your classroom location for lowest latency
   - **Pricing Plan**: Free tier (sufficient for 50+ concurrent users)
4. Click "Create new project" and wait 2-3 minutes for provisioning
5. Once project is ready, navigate to **Project Settings** → **API**
6. Copy the following values (we'll need them next):
   - **Project URL** (looks like https://xxxxx.supabase.co)
   - **Project API keys** → **anon public** key (long JWT token starting with eyJhbGci...)

Why manual: Supabase project creation requires account authentication and interactive dashboard setup. No CLI alternative exists for initial project creation.
  </instructions>
